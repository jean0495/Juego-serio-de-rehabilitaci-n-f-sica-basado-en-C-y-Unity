# Juego-serio-de-rehabilitaci-n-f-sica-basado-en-C-y-Unity
Este es un proyeto basado en el desarrollo de un juego serio para la rehabilitación fisica mediante el uso del motor gráfico Unity y la adquisición, procesamiento y envio de datos en tiempo real con lenguaje C. Todo lo anterior usando el sensor MPU 6050 el cual es un sensor incercial con giroscopio integrado para detetar movimientos de la muñeca y con estos mover el jugador en Unity. Por otro lado, se cuenta con el sensor Ad8232, el cual es un modulo para detectar señales ECG pero en el codigo se acondiciona para que sea EMG con el fin de capturar pulsos en el antebrazo y controlar el jugador mediante estos.
